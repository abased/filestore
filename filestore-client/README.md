murray-library
==============

the murray library is the foundation upon which all microservices are to be build


## IntelliJ Set-Up

File -> Open -> pom.xml

## Custom Properties

    /ets/sosacorp/email.client.properties

        smtp.host=localhost
        smtp.port=25


## Build

> mvn clean install
